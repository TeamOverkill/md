Eigen::Vector3d dimensionality¶

A vector which determines the translational degrees of freedom. In each time step the position of all atoms is multiplied with this vector. If the user specified parameter dimensions is set to 2, this vector will be [1, 1, 0] and hence will force all atoms to the xy-plane.
